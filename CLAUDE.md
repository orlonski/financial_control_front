# Financial Control Frontend

## Visao Geral
Aplicacao frontend para controle financeiro pessoal.

## Stack
- React com TypeScript
- Vite como bundler
- Tailwind CSS para estilos
- React Router para navegacao
- Axios para requisicoes HTTP

## Estrutura de Pastas
- src/components/ - Componentes reutilizaveis
- src/pages/ - Paginas da aplicacao
- src/hooks/ - Custom hooks
- src/services/ - Chamadas de API
- src/utils/ - Funcoes utilitarias
- src/types/ - Tipos TypeScript

## Comandos
- npm run dev - Servidor de desenvolvimento
- npm run build - Build de producao
- npm test - Rodar testes

## Padroes
- Componentes funcionais com hooks
- Tipos TypeScript para props
- Tailwind para estilos (evitar CSS custom)
- Nomes de componentes em PascalCase
- Nomes de arquivos em kebab-case ou PascalCase
