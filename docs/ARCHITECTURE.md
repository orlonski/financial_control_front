# Arquitetura do Frontend - Financial Control

## Visao Geral
SPA React para controle financeiro pessoal.

## Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router DOM
- Axios

## Estrutura
- src/components/ - Componentes reutilizaveis
- src/pages/ - Paginas/rotas
- src/hooks/ - Custom hooks
- src/services/ - API calls
- src/utils/ - Helpers
- src/types/ - TypeScript types

## Padroes
- Componentes funcionais
- Custom hooks para logica
- Context API para estado global
- Tailwind para estilos
